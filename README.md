# Crypto-DSA

DSA implementation with OpenSSL library in C++.

## Build

```bash
$ git clone https://github.com/Triple-Z/Crypto-DSA.git
$ cd Crypto-DSA/
$ cd src/
$ make
$ make run
```

> `OpenSSL`, `GNU Make` & `G++` required.

## Clean

```bash
$ make clean
```

## Debug

```bash
$ make debug
```

> `GDB` required.
